# Desktop Assistant (Jarvis)

## Description

A voice-controlled desktop assistant developed using Python.

## Features

- Open Google
- Open YouTube
- Search Wikipedia
- Tell the current time
- Voice response

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
